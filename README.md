# Caça Infinita 🧩

O **Caça Infinita** é um aplicativo Android moderno dedicado aos entusiastas de caça-palavras. O projeto foi desenvolvido com foco em escalabilidade, performance e uma experiência de usuário fluida, oferecendo desafios de forma infinita e dinâmica.

## 🚀 Sobre o App
O aplicativo permite que os usuários joguem caça-palavras de forma ilimitada, utilizando tecnologias de processamento de texto e inteligência artificial para melhorar a interação. 

### Principais Funcionalidades
*   **Geração Dinâmica:** Jogos gerados automaticamente para garantir que você nunca fique sem desafios.
*   **Reconhecimento de Texto:** Integração com ML Kit para identificar padrões e textos.
*   **Modo Offline:** Suporte a banco de dados local para jogar em qualquer lugar.
*   **Interface Moderna:** Design baseado em Material 3 com suporte a temas claro e escuro.

## 🛠 Tecnologias Utilizadas
O projeto segue as melhores práticas de desenvolvimento Android, utilizando uma arquitetura limpa (Clean Architecture) e modular:

*   **Linguagem:** [Kotlin](https://kotlinlang.org/)
*   **Interface (UI):** [Jetpack Compose](https://developer.android.com/jetpack/compose) com Material 3
*   **Injeção de Dependência:** [Hilt](https://developer.android.com/training/dependency-injection/hilt-android)
*   **Banco de Dados:** [Room](https://developer.android.com/training/data-storage/room)
*   **Inteligência Artificial:** [ML Kit](https://developers.google.com/ml-kit) (Text Recognition & Language ID)
*   **Câmera:** [CameraX](https://developer.android.com/training/camerax)
*   **Processamento de Documentos:** PDFBox e Apache Lucene
*   **Analytics & Crash:** Firebase (Analytics & Crashlytics)
